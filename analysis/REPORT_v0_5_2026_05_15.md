### 1. Arm B — 5-fold cell-stratified random county split

**Design:** SEED = 20260602, K = 5. Counties stratified by cell so every
cell appears in every fold. Fold sizes: 569 / 570 / 572 / 579 / 582
training units (held-out fold is sized accordingly).

For each fold k, the v0.4 model (negative-binomial GLM with HOLC + VRA +
sundown + cell-stratified random slopes) is re-fit on the four other folds
with seed `20260700 + k`, 4 chains × 1000 warmup × 1000 sampling, adapt_delta
0.95.

**Per-finding pre-reg pass criterion:** at least 4 of 5 fold posteriors must
have 95% CI clean on the correct side of zero.

### 3. Arm A — 2014–2018 outcome window — DEFERRED

**Original plan:** WISQARS 2014–2018 county-level firearm deaths,
identical model spec to v0.4, per-finding pass criteria of ±0.2 / ±0.3 /
±0.1 around the v0.4 reference coefficients, plus CI cleanness.

**Blockers encountered:**

1. **WISQARS API:** does not serve years before 2018. The `cdc_psx4_wq38`
   small-area-modeled dataset starts at 2019. Pivoted to CDC WONDER.
2. **CDC WONDER XML API:** initially rate-limited (15-sec floor between
   requests). Patched to 20-sec throttle. Second attempt surfaced multiple
   undocumented default-selection requirements that the XML schema does
   not satisfy on first POST:
   - `'ICD-10 113 Cause List' include both '#Malignant neoplasms (C00-C97)'
     and subordinate item(s)` — WONDER auto-includes a default 113-cause
     selection that conflicts with raw ICD-10 filter mode.
   - `No values were selected for Year/Month` — F_D76.V1 with explicit
     year list is rejected; requires interaction with the WONDER finder
     tool state machine.
   - `No values were selected for States` — F_D76.V10 default not set.
   - Two further finder-stage configuration messages.

**Status:** WONDER XML API requires reverse-engineering the full finder
state machine (the parameter schema is not fully documented; the canonical
approach is to construct the query via the WONDER web UI and capture
the resulting XML). Deferred until either: (a) a session is dedicated to
WONDER finder-tool replay, or (b) an alternative source (NBER MCD raw
mortality files, ≈2 GB per year) is fetched and processed.

**Decision:** Arm A is independently desirable (gives a pre-COVID time
window check) but not load-bearing for the v0.5 replication claim. Arm B
alone establishes that the three structural findings are robust to random
county sub-sampling at the n ≈ 570-per-fold scale.
